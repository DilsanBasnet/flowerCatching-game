# FlowerCathing Game
*It is Very Intresting web game where you have to collect flowers and avoid bomb to save you bucket and to score points..*
*It is build using html, css, & basic Js, In this game their are health bar, Score bar, a bucket and flowers, bombs . 

[**Live Demo**](https://flower-catching-game.vercel.app/) 

---
Simpel Rule: 
- Avoid bombs
- Collect Flowers
---
## Tech Stack
*   **Frontend:** HTML5, CSS3 
*   **Logic:**  JavaScript
